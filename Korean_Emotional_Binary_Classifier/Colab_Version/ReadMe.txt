=======================================<사용법>=======================================


한국어 감정 이진 분류기
Kor_Emo_Classifier.ipynb(코랩 버전)


1. 모델 파일은 패턴인식_프로젝트/소스코드_및_프로그램/Kor_Emo_Classifier/Local_Version/P_OR_N_MODEL 이름의 폴더를 사용한다.

2. 모델 파일(P_OR_N_MODEL)을 코랩에 업로드 한다.

3. 모델 파일의 경로를 코랩에서 입력하여 호출한다.

4. GUI 기반이 아니라는 부분을 제외하고 그 외 기능은 로컬 버전 과 동일하다.
